# plex-media-server

## Setup
Create directories:

```
mkdir /home/user/docker
mkdir /home/user/data

mkdir -p /home/user/data/torrents/tv
mkdir -p /home/user/data/torrents/movies
mkdir -p /home/user/data/media/tv
mkdir -p /home/user/data/media/movies
```

Copy the .env file:

```
cp .env.example .env
```
